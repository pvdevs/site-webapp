# SouJunior website

Repositório do site oficial da SouJunior: https://www.soujunior.tech/

![Logo](./logo-soujunior.png)

## 🚀 Quick Start
Instruções rápidas para rodar o projeto localmente.


**Instalação:**
```bash
  npm install
```

**Uso:**
```bash
npm run dev
```
